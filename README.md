Sistemas Empotrados:

**Diseño de Sistema Embebido en FPGA | Xilinx Vivado & VHDL**

Prácticas y proyecto final utilizando Vivado(VHDL) y Vitis(C)
para programar una placa Basys3 y sus diferentes periféricos.

- Desarrollo de una arquitectura SoC completa utilizando el microprocesador MicroBlaze sobre una placa FPGA Basys3.  
- Implementación y testeo de periféricos personalizados mediante interfaz AXI-Lite  
(conversor A/D, control de matriz de puntos y coprocesador dedicado). 
- Optimización del consumo de recursos y temporización del sistema mediante restricciones en Vivado.
